# 🏥 Noor Clinic & Maternity Center - Advanced Medical Portal

An advanced, responsive, and feature-rich healthcare management portal designed for  **Noor Clinic & Maternity Center** , G-11 Markaz, Islamabad. This project focuses on high-end UI/UX and seamless patient-doctor connectivity.

## 🚀 Key Features

### 1. **Smart Contact Routing (Auto-Shuffle)**

* **WhatsApp Integration:** Automatically routes chat queries to the clinic's mobile number (+92 333 5371687).
* **Landline Connectivity:** Direct display and call-link for the official landline (+92 51 2362908) to separate administrative calls from patient chats.

### 2. **Professional Administrative Highlights**

* **Verified Admin Profiles:** Features **Dr. Fatima Mashal** as the **Senior Administrator (NIRMS)** with custom verified badges and detailed professional background.
* **Strategic Branding:** Highlights the clinic's government-approved status and its association with senior medical officials.

### 3. **Advanced UI/UX Design**

* **Modern Aesthetics:** Implements  **Glassmorphism** ,  **Parallax effects** , and **Smooth Animations** (using Animate.css).
* **Interactive Services Hub:** 3D-hover effects on medical service cards (Emergency, Lab, Pharmacy, Maternity).
* **Fully Responsive:** Optimized for all screen sizes, from mobile devices to desktop monitors.

### 4. **Dynamic Doctor Management**

* **Real-time Updates:** Admin dashboard allows for easy CRUD (Create, Read, Update, Delete) operations for doctor listings.
* **Department Filtering:** Doctors are automatically grouped by their specializations (e.g., Gynecologist, General Physician).

## 🛠️ Technology Stack

* **Backend:** PHP / Laravel Framework.
* **Frontend:** Bootstrap 5, FontAwesome 6, Animate.css.
* **Database:** MySQL for structured medical data management.
